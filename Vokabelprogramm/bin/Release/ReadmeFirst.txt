können Sie den Ordner auch über die Applikation erzeugen lassen, indem Sie sie starten und wieder 
beenden.Entpacken Sie sodann die beigefügten Listen der Datei 'Listen.zip' in den soeben erstellten 
Ordner.Ab jetzt stehen Ihnen drei Demolisten zum Test zur Verfügung. Selbstverständlich können sie diese
Listen abändern oder aber neue erstellen. Demnächst werden noch mehr Listen in folgenden Sprachen
auf meinem Server gehostet: Englisch, Französisch, Spanisch, Russisch. Besuchen Sie meine WebSite unter:
